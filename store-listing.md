## Permissions Explanation (for Play Store review)
- READ_CALENDAR: Displays the user's next calendar event on screen
- RECORD_AUDIO: Listens for configurable emergency keyword for voice-activated emergency calls
- INTERNET: Sends emergency API call when voice trigger is detected
- ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION: Calculates real sunrise/sunset times based on device location
- WAKE_LOCK: Keeps screen on for always-on display functionality
- FOREGROUND_SERVICE: Required for continuous voice recognition service
